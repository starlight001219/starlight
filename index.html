站点怎么玩

  • 先当选手：不看源码、不看提示，尽量拿到每一题的第一层 flag。
  • 再当出题人：右键查看源码、读注释、翻隐藏节点，寻找设计陷阱。
  • 最后当审计员：检查哪些 flag 是蜜罐，哪些才是真正答案。
  • 隐藏彩蛋：在任意题目输入框输入 starlight{help} 试试。

出题人提示

  • 直接把整个目录压缩分享即可，不需要服务器。
  • 若要线上办赛，可直接部署到 GitHub Pages / Vercel / Netlify。
  • 每道题页面都提供提交 flag 入口，方便现场核验。
  • 这三道题不是孤岛，它们之间有隐藏的线索传递。回溯是一种美德。
