密信不会自己开口

这条密信是一条加密版的网络黑话。有人说要跑脚本、算密钥，但真相是：你只要在脑海里默念这题有毒三遍，然后去前面的题里找找被人遗忘的话。而且，你手里的碎片可能比你以为的多。

Base64跨题碎片星芒密钥伪装成功

密文

U2FsdGVkX19Zb3VyU2VjcmV0UGhyYXNlCg==

提示：这段看起来像 Base64，但它还不是终点。解码后你会发现一个关键词，它不是 flag 本身，而是通往真相的引子。

// 裁判侧记： // 碎片 A：edebc9edd9ded3dcdddfd5dad7dec8 // 碎片 B：d6e6dae2e0d8c6cae9e7d3d4d3d9ce // 碎片 C：d2dcd6dfcdd7c6d4d5ededc0ddd1e2e4 // 密钥：starlight // 拼接顺序在前三题的蜜罐里。 // 解密方法：Vigenere 反向偏移。

碎片解密器

输入你从蜜罐里收集到的 hex 碎片（不含空格），用 starlight 密钥解密。

输入碎片，点击解密。

碎片收集指南

  1. 你可能已经在前两题里收集到了碎片，只是没意识到它们是碎片。
  2. 真正的 flag 被打散成了三段密文，分散在站点各处的蜜罐里。
  3. 密钥是站点的名字：starlight。用 Vigenere 反向偏移即可解密。
  4. 把三段解密结果按顺序拼起来，就是最终 flag。

如果你手里的碎片不够，回去翻翻前两题的蜜罐和注释。蜜罐不只是陷阱，它还是碎片容器。
